# Raspbot-Finding_Lane
only_Run这个文件能支持小车的运动，里面也有简单的循迹（但效果好像不太理想:(
简单逻辑是i2c通信控制小车电机转动和舵机转动，将摄像头画面视频流处理，同时每帧画面二值化处理（车道为白色）
通过遍历每列像素值得到白色像素（255）的数量，大于某一数量则认为查找到车道边缘，另一边同理，然后去中间值作为车道中点并画垂直线
将此线与摄像画面中线作比较，差值作为偏差值，本来想的是通过差值来控制转弯幅度，但是转不过来（车道太小，幅度一大每次转的角度特别大，不稳定）
于是想直接写死左右电机转动速度，并让一边轮子倒转来小转弯半径，还没试验
